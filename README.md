# Gaia Commons Council App (gaiaCC7)

React + Express + TypeScript + Vite full-stack app with Docker deploys.

## Quick Start

```bash
# Clone & install
git clone https://github.com/YOURUSER/gaiaCC7.git
cd gaiaCC7
npm install

# Dev (client:3000, server:3001)
npm run dev

# Build & preview
npm run build
npm run preview
```

## Docker

```bash
docker compose up --build
```

## CI/CD
GitHub Actions: Build/Test on PR, auto-deploy main branch.
